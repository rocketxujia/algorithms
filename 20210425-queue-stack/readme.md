
## 堆栈 stack 
- 操作原则：后进先出
    原堆栈: 3 > 2 > 1
    栈顶为 3， 栈尾为 3。 
    栈顶进，栈顶出

- 操作方法：
    push(4):  4 > 3 > 2 > 1
    pop(): return 4, 堆栈变成 3 > 2 > 1
    top() / peek(): 查看栈顶元素 3
    clear(): 清空栈
    size(): 栈大小
    empty(): 栈是否为空

## 队列 queue
- 操作原则：先进先出
    原队列： 1 -> 2 -> 3
    队头：1
    队尾：3
    队尾进  1 -> 2 -> 3 -> 4 
    队头出  2 -> 3 -> 4 

- 操作方法：
    push(4):  1 -> 2 -> 3 -> 4 
    pop(): return 1, 队列变成 2 -> 3 -> 4 
    front(): return 1 查看队头元素
    back(): return 4 查看队尾元素
    size(): 栈大小
    empty(): 栈是否为空
